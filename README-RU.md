# [ENGLISH README](https://github.com/ILCKR/DDS-Resize-Tool/blob/main/README.md)

# Что это?

- **DDS Resize Tool** - инструмент для массового быстрого уменьшения и пережатия DDS текстур. Написан на обычном Powershell на базе **Texconv**
- Можно использовать для уменьшения текстур до 16 раз
- Лучше всего подходит для текстур, которые имели одинаковые настройки при их создании изначально

# Требования

1. **Powershell 7** - [Microsoft](https://learn.microsoft.com/ru-ru/powershell/scripting/install/install-powershell-on-windows?view=powershell-7.5)
2. **Texconv** - [Latest](https://github.com/Microsoft/DirectXTex/releases/latest/download/texconv.exe)

# Использование

1. Открыть `.ps1` файл через Powershell 7
    1. Скрипт автоматически проверяет наличие `Texconv.exe` рядом с собой - если не находит - просит перетащить `Texconv.exe` в окно Powershell для начала работы
2. Перетащить папку в которой лежат текстуры под пережатие
3. Выбрать степень уменьшения (2, 4, 8, 16 раз)
4. Подождать (процесс отображается)
5. По окончанию процесса откроется папка с преобразованными текстурами

### Важные нюансы при использовании

1. Настройки зашиты +- универсальные за одним большим НО: **КАЖДАЯ ТЕКСТУРА МОЖЕТ ТРЕБОВАТЬ СВОИ ИНДИВИДУАЛЬНЫЕ ПАРАМЕТРЫ AT И ФИЛЬТРОВ**
2. Все настройки, которые использует Texconv прописаны в строке **112**
    ```... tc -nologo -if FANT -f $fmt -m 0 ...```
3. Скрипт изначально умеет различать Bump и Diffuse текстуры.
4. Стандартные форматы сжатия: **BC1** для всех `.dds`, кроме `bump.dds` - для них по-умолчанию стоит **BC3**
